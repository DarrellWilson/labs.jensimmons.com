Get a Browser with CSS Grid

  • Firefox Nightly
  • Firefox CSS Grid Visualizer
  • Safari Technical Preview
  • To turn on Grid in regular Firefox, go to about:config, search for layout.css.grid.enabled, and click on false to turn it to true.
  • To turn on Grid in any version of Chrome or Opera, go to chrome://flags/#enable-experimental-web-platform-features or opera://flags/#enable-experimental-web-platform-features and click "Enable".
  • To turn on Grid in Safari Technical Preview, use the Develop menu to go to Experimental Features > CSS Grid. There's no way to test in regular Safari.
